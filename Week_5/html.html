How The Web Works

This unit was significantly different than the last one. While there was lots of hand holding in C, the unit on web languages has been fast pased and more 'think like a programmer'. Which is exactly what you would expect from this course.

Having a solid grasp of thses languages coming in means that, while I may have felt a little overwhelmed had I not had any experience, I have been free to explore and really just cement the basics. Which everyone needs from time to time.

HTML

This was the first of the languages that we picked up in this section of the course. While not a programming language, this structural language is the cornerstone of the web and whithout it, almost nothing else works.

Main concepts:

  1. Structure
  2. Semantic Tags
  3. How web pages are built

CSS

CSS is the style of the web. Simple and declairative. Choose a selector, choose a property, choose a value. If you know how key value pairs work you are good to go.

Main concepts:

  1. Properties
  2. Selectors
  3. Frameworks (Bootstrap)

JavaScript

The holy grail of web development. From front to back. If you know JavaScript you know the web. Now many pieces of the backend can be built with other languages, in fact this course uses python to do so. But when push comes to shove, JavaScript owns the hearts of many (most) web developers.

Main concepts:

  1. Website interaction
  2. The DOM
  3. Funtional Pages
