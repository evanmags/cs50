C (the language)

C was the first 'unit' per se of this course. While expansive, and covering several weeks all in all it can be grouped under the C umbrella.

Having come from a background in HTML, CSS and JavaScript the limitations of c were plainly obvious at first. Only they werent quite limitations, rather Structure that allowed for more understanding of what was going on under the hood.

See, C, being a low level language, meaning the programer works directly with memory,forces your to think about how every aspect of your application is being processed. Everything from the size of an array to the space needed to hold a piece of information needs to be considered to allow the program to run smoothly.

In less structured languages, like Python and JavaScript, types, memory allocation, sizes of variables dont matter because the kanguage will make an educated guess as to what you are trying to do.

Main concepts:

  1. Typing
  2. How data is stored in memory
  3. Data Types:
    • Integers
    • Floats
    • Doubles
    • Longs
    • Strings (char*)
    • Arrays
    • Structs
    • Linked Lists (single and double)
    • Hashtables
    • tries
  4. and much more!
